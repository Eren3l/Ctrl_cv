# 历史粘贴板 📋

一款 Windows 桌面小工具——自动记住你复制过的文字和图片，随时找回。

## 功能

- **自动记录** — 打开软件后，复制的文字和图片会自动保存
- **随时查看** — 列表按时间排列，点击查看完整内容
- **搜索筛选** — 输入关键词搜索，按日期范围筛选
- **重要内容置顶** — 右键置顶，置顶内容永不过期
- **过期自动清理** — 超过设置天数的内容提醒清理

## 使用

| 操作 | 方法 |
|------|------|
| 搜索 | 在顶部搜索框输入关键词 |
| 筛选日期 | 点搜索框旁边的下拉框选择时间范围 |
| 查看详情 | 点击列表中的任意一条 |
| 复制回剪贴板 | 展开详情后点「复制」按钮 |
| 置顶 / 取消置顶 | 右键某条记录 → 置顶 |
| 删除 | 右键某条记录 → 删除 |
| 设置保留天数 | 点右上角「设置」→ 选择天数 → 保存 |

## 注意

- 软件需要保持运行才会记录内容，关闭即停止
- 所有文字（包括密码等敏感信息）都会被记录，请自行删除
- 图片会自动压缩到 100KB 以内，不占空间
